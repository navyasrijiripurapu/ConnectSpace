# Connect Space
Connect Space is a social media platform that allows users to connect, share posts, interact with others, and build their online community. The application provides a modern and user-friendly interface with features similar to popular social networking platforms.
## Features
- User Registration and Login
- User Profiles
- Create, Edit, and Delete Posts
- Like and Comment on Posts
- Follow and Unfollow Users
- Search Users and Posts
- Real-Time Messaging
- Responsive Design
- Secure Authentication
## Tech Stack
### Frontend
- React.js
- HTML
- CSS
- JavaScript
- Material UI
### Backend
- Node.js
- Express.js
### Database
- MongoDB
### Additional Tools
- Socket.io (Real-Time Communication)
- JWT Authentication
